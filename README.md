# Dockerize Banking Application

This project composes two services:
1. Web Application
2. MySQL Database

Each service runs in a separate Docker container.

### Instructions:
#### Starting the Banking API server
1. `docker compose up`
2. Use the [sample client](https://github.com/satishsuryanarayan/banking-client) to interact with the Banking API server

#### Shutting down the Banking API server
1. `docker compose down`
